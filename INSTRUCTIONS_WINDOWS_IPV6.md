When running BikeControl and MyWhoosh on the same Windows device, some users report that OpenBikeControl connection cannot be established. This is a MyWhoosh issue, and they are determined to fix this in an upcoming release. In the meantime, users can resolve the issue by disabling IPv6 from their Wi-Fi settings using the steps below:

## Option 1: Via Control Panel
1. Open Control Panel
2. Navigate to **Network and Internet**
3. Click on **Network and Sharing Center**
4. Select **Change adapter settings**
5. Right-click on the Wi-Fi network and choose **Properties**
6. Locate **Internet Protocol Version 6 (IPv6)**
7. Uncheck the option and click **OK**

## Option 2: Quick Access 
1. Press **Windows + R**
2. Type `ncpa.cpl` and press Enter
3. Right-click on the Wi-Fi network and choose **Properties**
4. Locate **Internet Protocol Version 6 (IPv6)**
5. Uncheck the option and click **OK**
